# CBDパーソナル診断ツール - 診断ロジック案比較

## 前提条件

### 目標
- **CVR 10%達成**（現在の1-2%から5-10倍向上）
- **1日1件成約** → 月30件 → 固定費（約10万円）を2-3ヶ月で回収

### 設計思想
- ユーザーの趣味（ポーカー、マラソン、お酒）を絡めた「意外性のある選択肢」
- データコンサルらしい「論理的解決策」の提示
- 「笑える」「面白い」要素でエンゲージメント向上

---

## 案1: 「ライフスタイル×悩みマトリクス」型診断

### コンセプト
「あなたの趣味シーンで、どんな悩みが一番深刻？」を軸に、3ステップで商品を絞り込む。

### 診断フロー
```
STEP 1: メインの趣味を選択
└─ ポーカー / マラソン / お酒 / その他（仕事・勉強等）

STEP 2: 具体的な悩みを選択（趣味別に分岐）
├─ ポーカー → 「ティルトしやすい」「集中力が続かない」「疲れで判断力低下」
├─ マラソン → 「走った後の回復が遅い」「膝が痛む」「睡眠の質が悪い」
└─ お酒 → 「二日酔いがひどい」「飲み会で眠れない」「翌日が仕事で辛い」

STEP 3: 使用シーンを選択
└─ 「今すぐ必要」「毎日使いたい」「イベント前日に備えたい」
```

### 商品マッピング例
| 趣味 | 悩み | シーン | おすすめ商品 |
|------|------|--------|--------------|
| ポーカー | ティルト | イベント前日 | 高濃度CBDオイル（冷静さ特化） |
| ポーカー | 集中力 | 今すぐ | CBDグミ（即効性） |
| マラソン | 回復 | 毎日 | CBDリカバリークリーム |
| お酒 | 二日酔い | 今すぐ | CBDドリンク |

### メリット
✅ **シンプルで分かりやすい** - 3ステップで迷わない  
✅ **趣味に直結** - ポーカー・マラソン・お酒の要素を明確に活用  
✅ **CVR見込み:** 8-12%（理由：悩みが明確→商品選びが簡単→購入意向が高い）

### デメリット
❌ 意外性がやや弱い（王道すぎる可能性）  
❌ 複数の悩みを抱えるユーザーには物足りない

## 案2: 「緊急度×シチュエーション特化」型診断

### コンセプト
「データコンサルが分析した、あなたの"今"に最適なCBD」という切り口。具体的なシチュエーションと緊急度で判断。

### 診断フロー
```
STEP 1: 今どんな状況？（シチュエーション選択）
├─ 「ポーカー大会が明日」「マラソン大会が明後日」「今夜飲み会」
├─ 「ポーカーで全ロスして落ち込んでる」「フルマラソン完走したばかり」
└─ 「仕事で集中力が必要」「勉強の効率を上げたい」（汎用）

STEP 2: 緊急度は？（データコンサル風に数値化）
└─ 「今すぐ必要（緊急度10/10）」「3日以内（緊急度7/10）」「予防的に（緊急度3/10）」

STEP 3: 過去の経験（分析要素）
└─ 「CBD使ったことある？」「サプリメント経験は？」「即効性重視？継続重視？」

→ 結果：「あなたの状況に最適なCBD戦略」として商品を提示
```

### 商品マッピング例
| シチュエーション | 緊急度 | 結果 |
|----------------|--------|------|
| ポーカー大会明日 | 10/10 | 「大会前夜の集中力UPセット（高濃度CBD + 睡眠サポート）」 |
| 全ロスで落ち込み | 10/10 | 「ティルトリセットCBD（リラックス特化型）」 |
| フルマ完走直後 | 7/10 | 「筋肉回復CBD + 睡眠サポート」 |
| 今夜飲み会 | 7/10 | 「飲む前・飲んだ後セット（予防+回復）」 |
| 予防的に | 3/10 | 「日常使いCBDセット（コスパ重視）」 |

### メリット
✅ **意外性が高い** - 「ポーカーで全ロス」など具体的なシチュエーション  
✅ **データコンサルらしさ** - 緊急度の数値化、「戦略」という言葉  
✅ **エンゲージメント高** - ユーザーが「あるある」と共感しやすい  
✅ **CVR見込み:** 12-15%（理由：シチュエーションが具体的→購入動機が強い）

### デメリット
❌ 分岐が複雑になりやすい  
❌ シチュエーションのパターンを増やす必要がある

## 案3: 「性格×集中力3軸分析」型診断

### コンセプト
「あなたの集中力・回復力・リラックス力を3軸で分析」というデータコンサルらしいアプローチ。ポーカー（集中）、マラソン（持久・回復）、お酒（リセット）の3軸。

### 診断フロー
```
STEP 1: 集中力チェック（ポーカー軸）
Q1. ポーカーや集中が必要な作業で、どれくらい集中力が続く？
Q2. ミスをした時に、どれくらい落ち込む？（ティルト度）

STEP 2: 持久力・回復力チェック（マラソン軸）
Q3. 運動後の回復は早い方？
Q4. 睡眠の質はどう？（翌朝のスッキリ感）

STEP 3: リセット力チェック（お酒軸）
Q5. 二日酔いはどれくらいで回復？
Q6. ストレス発散は何で？（お酒以外も選択可）

→ 結果：3軸のレーダーチャートを表示
   「あなたは【集中力特化型】【回復力特化型】【バランス型】です」
   → タイプ別に商品を提案
```

### タイプ別マッピング
| タイプ | 特徴 | おすすめ商品 |
|--------|------|--------------|
| **集中力特化型** | ポーカー軸が高い | 高濃度CBDオイル（集中力UP） |
| **回復力特化型** | マラソン軸が高い | CBDリカバリーセット |
| **リセット特化型** | お酒軸が高い | CBDドリンク + 睡眠サポート |
| **バランス型** | 3軸とも中程度 | オールラウンドCBDセット |

### メリット
✅ **データコンサルらしさMAX** - レーダーチャート、3軸分析  
✅ **視覚的に面白い** - 結果がグラフで見える  
✅ **再診断したくなる** - 結果をシェアしたくなる設計  
✅ **CVR見込み:** 10-13%（理由：タイプ診断の納得感→商品選びがスムーズ）

### デメリット
❌ 実装がやや複雑（グラフ表示が必要）  
❌ 質問が6問とやや多い（離脱の可能性）

## 推奨案

### 🏆 **案2: 緊急度×シチュエーション特化型**

**理由:**
1. **CVR見込みが最高（12-15%）** - シチュエーションが具体的で購入動機が強い
2. **意外性が高い** - 「ポーカーで全ロス」などユーザーの価値観（お笑い×意外性）に合致
3. **エンゲージメントが高い** - 「あるある」で共感されやすい
4. **実装難易度が適切** - 案3よりシンプルで、案1より面白い

### 実装時の調整案
- **案2をベースに、案3の要素を一部取り入れる**
  - 結果表示時に「あなたの緊急度スコア: 10/10」など数値化
  - シンプルなプログレスバーで視覚化（レーダーチャートまでは不要）
