## 📝 修正内容の詳細

### 1️⃣ 段落の最大行数を2行に制限

**❌ 旧ルール:**
```
3行以上の段落もOK
```

**✅ 新ルール:**
```html
<!-- NG例：3行以上 -->
<p>2026年現在、日本においてTHC（テトラヒドロカンナビノール）は、「大麻取締法」によって厳しく規制されている違法物質です。
この規制は、THCが持つ精神作用（いわゆる「ハイになる」効果）が依存性や乱用につながるリスクがあるため、
国民の健康と安全を守る目的で設けられています。</p>

<!-- OK例：最大2行 -->
<p>2026年現在、日本においてTHC（テトラヒドロカンナビノール）は、「大麻取締法」によって厳しく規制されている違法物質です。<br />
この規制は、THCが持つ精神作用によるリスクを防ぐため、国民の健康と安全を守る目的で設けられています。</p>
```

### 2️⃣ 「この記事で分かること」のコントラスト改善

**❌ 旧デザイン:**
```html
<div style="background-color: #f0f7f0; border-left: 5px solid #2d7a4c; padding: 16px; margin: 20px 0;">
  <p style="font-weight: bold; margin: 0 0 12px 0; color: #2d7a4c;">✓ この記事で分かること</p>
```
→ 緑背景＋緑テキスト＝見にくい

**✅ 新デザイン:**
```html
<div style="background-color: #f0f7f0; border-left: 5px solid #2d7a4c; padding: 16px; margin: 20px 0;">
  <p style="font-weight: bold; margin: 0 0 12px 0; color: #333;">✓ この記事で分かること</p>
```
→ 緑背景＋黒テキスト＝見やすい

### 3️⃣ エビデンス・参考リンクセクションの追加

**位置:** 記事の最下部（まとめの後、関連記事の前）

**デザイン:**
```html
<div style="background-color: #f5f5f5; border-top: 2px solid #2d7a4c; padding: 16px; margin-top: 20px;">
  <p style="font-weight: bold; margin: 0 0 12px 0; color: #2d7a4c;">📚 参考リンク・エビデンス</p>
  <ul style="margin: 0; padding-left: 20px; font-size: 14px;">
    <li><a href="URL1">出典1：説明</a></li>
    <li><a href="URL2">出典2：説明</a></li>
    <li><a href="URL3">出典3：説明</a></li>
  </ul>
</div>
```

**特徴:**
- ✅ グレー背景（#f5f5f5）
- ✅ 上ボーダー（#2d7a4c）
- ✅ 各記事に3～5個のURLを含める
- ✅ 公式サイト、論文、参考記事へのリンク

### 4️⃣ 1見出し内の文字数を厳密化

**現在のルール:** 150-200字以内 → **実際には超過している**

**新ルール:**
- 絶対上限：150字
- 超過したら **H4で細分化 or セクション削除**
- 厳密にチェック

### 5️⃣ 参考記事1097のシンプルさを実現

**分析（予想）:**
- 段落数：多い（20-30個以上）
- 平均段落長：短い（50-80字程度）
- 見出し数：多い（5-7個程度）
- ビジュアル：表・リスト・ボックスを効果的に配置

**実装ルール:**
```
- 1段落＝最大2行
- 1段落＝50-80字目安（最大100字）
- 見出し＝4-5個推奨
- 短い文で、改行を活用
- ビジュアル要素で区切る
```
