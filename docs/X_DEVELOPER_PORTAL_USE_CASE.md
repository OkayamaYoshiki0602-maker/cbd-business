## Xのデータと API の使用例説明（日本語）

### 1. プロジェクト概要

**プロジェクト名:** CBD情報サイト運営・自動投稿システム

**目的:** CBD（カンナビジオール）に関する情報提供サイト「CBD WORLD」（https://cbd-no-hito.com/）の運営を効率化し、ユーザーに有益な情報を継続的に提供するための自動化システムを構築します。

---

### 2. 具体的な使用例

#### 2.1 サイト記事の自動投稿

**目的:**
WordPressで公開された新着記事を、Xで自動的に共有してサイトへのトラフィックを増やす。

**実装方法:**
- WordPressのRSSフィードまたはWebhookを使用して新着記事を検知
- 記事タイトル・要約から、280文字以内のツイート文案を自動生成
- 記事URLと共にXに投稿
- 投稿前にLINE通知で内容を確認（人的確認）

**使用するAPI:**
- `POST /2/tweets` - ツイート投稿
- `POST /1.1/media/upload` - 画像アップロード（記事サムネイル）

**投稿頻度:**
- 新着記事公開時のみ（通常1日0-2回程度）
- スパム行為は行わない
- レート制限を遵守

**コンテンツ:**
- 記事タイトルと要約
- 記事URL
- 関連ハッシュタグ（#CBD #健康 #睡眠 など）
- サイトブランド名（「CBD WORLD」）

---

#### 2.2 CBD関連ニュースの共有

**目的:**
CBD業界の最新ニュースや研究結果を、ユーザーに迅速に共有する。

**実装方法:**
- 信頼できるCBD情報源（研究機関の公式サイト、医療情報サイトなど）のRSSフィードを監視
- 新しいニュースを検知したら、要約と出典URLを含むツイートを自動生成
- 投稿前にLINE通知で内容を確認（人的確認）
- ニュースの正確性を担保するため、医療・健康に関する内容は必ず専門家による確認を行う

**使用するAPI:**
- `POST /2/tweets` - ツイート投稿

**投稿頻度:**
- ニュース公開時のみ（通常1日0-3回程度）
- 情報の質を重視し、スパム行為は行わない

**コンテンツ:**
- ニュース要約（280文字以内）
- 出典URL
- 関連ハッシュタグ
- サイトブランド名

---

#### 2.3 診断ツールの告知

**目的:**
サイト内の「CBDパーソナル診断ツール」を、適切なタイミングでXで告知する。

**実装方法:**
- 診断ツールの機能や特徴を紹介するツイートを定期的に投稿
- ユーザーの悩み（睡眠、ストレス、集中力など）に応じた誘導文を作成
- 診断ツールのURLと共に投稿

**使用するAPI:**
- `POST /2/tweets` - ツイート投稿

**投稿頻度:**
- 週1-2回程度（過度な投稿は避ける）
- ユーザーエンゲージメントを重視

---

#### 2.4 ユーザーエンゲージメント向上

**目的:**
Xでのフォロワー（現在約1,000人）とのコミュニケーションを維持し、サイトへのリピーターを増やす。

**実装方法:**
- サイトのアクセス解析データ（Google Analytics 4）を活用して、人気記事やトレンドを把握
- ユーザーが興味を持ちそうなトピックを選定
- 人間が作成したツイート文案を、適切なタイミングで投稿
- 自動リプライやスパム行為は行わない

**使用するAPI:**
- `POST /2/tweets` - ツイート投稿
- `GET /2/tweets/search/recent` - リプライやメンションの検索（将来的に）

---

### 3. データの使用方法

**取得するデータ:**
- ツイート投稿結果（投稿ID、URLなど）
- ツイートのエンゲージメント（いいね数、リツイート数など） - 将来的に
- ユーザー情報（自分のアカウント情報のみ）

**データの用途:**
- 投稿履歴の記録
- エンゲージメント分析（投稿タイミングの最適化）
- サイトへのトラフィック効果測定

**データの保存・共有:**
- 投稿履歴はローカル環境のみに保存
- 外部にデータを再販売することは一切ありません
- ユーザーのプライバシーを尊重し、個人情報の収集は行いません

---

### 4. 自動化ルールの遵守

**遵守する事項:**
- ✅ スパム行為は行わない（過度な投稿を避ける）
- ✅ 自動リプライや自動メンションは行わない
- ✅ 投稿内容は人間が確認した上で投稿する
- ✅ Xの利用規約・開発者規約を遵守する
- ✅ レート制限を遵守する
- ✅ ユーザーのプライバシーを尊重する

**自動化の範囲:**
- 記事更新検知、ツイート文案生成、投稿予約までは自動化
- **投稿実行前には必ずLINE通知で人間が確認**
- 不適切な内容やスパムと判断される内容は投稿しない

---

### 5. 商業利用について

**商業目的:**
- CBD商品のアフィリエイトサイト運営
- サイト内のアフィリエイトリンクから収益を得る
- Xでのツイートは、サイトへのトラフィック誘導が目的

**収益モデル:**
- アフィリエイトリンクからの手数料（サイト内）
- X APIを使用した直接的な収益化は行わない
- Xのデータを再販売することは一切ありません

---

### 6. 技術的な実装

**開発環境:**
- Python 3.9+
- tweepy ライブラリ（X API v2）
- WordPress REST API（記事更新検知）
- LINE Messaging API（投稿前確認）

**運用方法:**
- ローカル環境またはプライベートサーバーで実行
- GitHubでコード管理
- 定期的なメンテナンスと更新

**セキュリティ:**
- API認証情報は環境変数で管理（.envファイル）
- 認証情報はGitにコミットしない
- 定期的なセキュリティチェック
