# DMMアフィリエイトコンプライアンスチェックリスト

## 📋 DMMウェブサービス利用規約の重要ポイント

### ✅ 必須遵守事項

#### 1. リンク設置（第8条2項）
- ✅ **必須**: DMMへのリンクを設置する（アフィリエイトリンクで対応）
- ✅ **禁止**: ウェブサービスを使用した部分でDMM以外のリンクを設置しない
  - ツイート本文内にアフィリエイトリンクのみ設置する

#### 2. クレジット表示（第11条）
- ✅ **推奨**: 商品情報がDMMによって提供されたものであることを表示
  - プロフィールに「アフィリエイトリンクを含みます」と明記（実質的なクレジット表示）

#### 3. 禁止行為（第9条1項）
- ✅ **遵守**: 以下の内容を含まない
  - 法令または本規約に違反する内容
  - 知的財産権を侵害する内容
  - 公序良俗に反する内容
  - **不潔またはグロテスクなコンテンツ等一般人が不快感を覚える内容**
  - 不当な表示
  - 閲覧者に誤解を与えるおそれのある内容
  - **「サイトの運営のためにクリックしてください」等閲覧者に対しクリックを依頼または嘆願する内容**
  - 当社との提携関係を表示しまたは暗示する内容

## 🔍 コンプライアンスチェック

### Twitterアカウント設定

- [ ] プロフィールに「18歳未満閲覧禁止」を明記
- [ ] プロフィールに「アフィリエイトリンクを含みます」を明記
- [ ] 過度に露骨なプロフィール画像を使用していない
- [ ] R18表記（🔞）を明記

### ツイート内容

- [ ] サムネイル画像は控えめ（過度に露骨でない）
- [ ] 「クリックしてください」等の依頼・嘆願をしていない
- [ ] DMM以外のリンクをアフィリエイトリンクと一緒に掲載していない
- [ ] 商品情報を正確に表示
- [ ] 誤解を与える表現を使用していない

### 技術的遵守

- [ ] DMMアフィリエイトプログラムを使用してリンクを作成
- [ ] API IDを適切に管理（.envファイルで管理）
- [ ] 商品情報を適切な目的以外で使用していない

---

## ⚠️ 特に注意すべきポイント

### 1. 画像の選択

**OK**: 
- 公式サムネイル画像
- 控えめなサンプル画像
- モザイク処理済み画像

**NG**:
- 過度に露骨な画像
- 無修正画像
- 一般人が不快感を覚える画像

### 2. ツイート文面

**OK**:
- 「詳細・購入⇩」
- 「続きを読む⇩」
- 商品情報の紹介

**NG**:
- 「クリックしてください」
- 「サイトの運営のためにクリックしてください」
- 当社との提携関係を暗示する表現（「公式」「推奨」など）

### 3. リンクの扱い

**OK**:
- DMMアフィリエイトリンクのみ

**NG**:
- DMM以外のアフィリエイトリンク（同じツイート内）
- 他のアフィリエイトプログラムへのリンク
