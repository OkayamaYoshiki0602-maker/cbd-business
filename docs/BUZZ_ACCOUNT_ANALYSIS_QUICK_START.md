## 💡 分析のコツ

### 効率的に進めるポイント

1. **最初に各アカウントのプロフィールを確認**
   - フォロワー数、ツイート数、自己紹介文を確認

2. **最新20件程度のツイートをスクロール**
   - ツイート構成、文字数、改行の有無を確認
   - アフィリエイトリンクを含むツイートの割合を確認

3. **バズツイート（いいね数が多いもの）を重点的に確認**
   - バズツイートの共通点を見つける
   - 文字数、構成、タイミングなどを記録

4. **数値は概算でOK**
   - 完璧な数値ではなく、傾向がわかれば十分
   - 例: 「1日5-8回」「文字数は多め（200文字前後）」
