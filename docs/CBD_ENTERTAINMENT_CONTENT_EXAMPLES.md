# CBDエンタメ×情報型の具体例

## 🎯 コンセプトの整理

「エンタメ×情報型」とは、**「面白おかしくする」のではなく、「面白く情報を伝える」**ことです。

CBDという健康・医療関連のジャンルを扱う場合、以下のバランスが重要：
- ✅ **情報の正確性・信頼性を保つ**
- ✅ **伝え方を面白く・意外性を持たせる**
- ✅ **ユーザーの体験（ポーカー、マラソン、お酒）を絡める**
- ✅ **データコンサルタントの視点を適度に活用**

---

## 🎨 エンタメの「程度」について

### ❌ やりすぎるエンタメ（避けるべき）

```
【CBDで超能力が覚醒した話】
【CBDを飲んだら時間が止まった】
【CBDで宇宙人と会話できた】
```

**問題点**:
- 信頼性が失われる
- 医療・健康関連のジャンルに不適切
- アフィリエイト商品への信頼が失われる

---

### ✅ 適切なエンタメ（推奨）

```
【CBDを試した結果、予想外の副作用が...（実体験ベース）】
【データ分析した結果、意外なことが判明（データベース）】
【失敗談から学んだ、正しいCBDの選び方（学びベース）】
```

**ポイント**:
- 実体験・データ・学びをベースにする
- 情報の正確性を保つ
- 意外性・面白さは「伝え方」で表現する
- 信頼性を損なわない範囲でエンタメ性を出す

## 💡 「エンタメ×情報型」の本質

### エンタメ = 「面白おかしくする」ではない

**エンタメ = 「面白く情報を伝える」**

- ✅ **情報の正確性・信頼性を保つ**
- ✅ **伝え方を面白く・意外性を持たせる**
- ✅ **ユーザーの体験を絡める**
- ✅ **データ分析の視点を適度に活用**

## 📊 他のパターンとの比較

### パターン1: データ×CBD型（専門性重視）
```
【CBDの研究データを分析した結果、〇〇が判明】
→ 信頼性高いが、エンタメ性が低い
```

### パターン4: エンタメ×情報型（バランス型）
```
【CBDを試した結果、予想外の副作用が...（データ分析も含む）】
→ 信頼性を保ちつつ、エンタメ性も加える
```

**違い**:
- パターン1: 情報中心、エンタメ性低い
- パターン4: 情報 + エンタメ性（伝え方を工夫）
