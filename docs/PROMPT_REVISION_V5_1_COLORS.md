# 記事生成プロンプト修正 v5.1 - 色使い・チェックリスト統一

## 🎨 最終的な色使い統一

### 参考イメージ（添付）の分析：

**緑系で統一された配色:**
- ✅マーク：緑色（#2d7a4c）
- テキスト：黒（#333）または濃い緑（#2d7a4c）
- 背景：ライトグレー（#f5f5f5）または薄緑（#f0f7f0）
- ボーダー：緑（#2d7a4c）

**特徴:**
- 色数を最小化（緑系のみ）
- 赤色やオレンジは使用しない
- 統一感がある、落ち着いた印象

---

## 🎯 修正内容

### 1️⃣ Q&Aのテキスト色修正

**❌ 旧デザイン:**
```html
<p style="margin: 0; color: #e74c3c; font-weight: bold;">A.</p>
```
→ 赤色でカテゴリーが異なる

**✅ 新デザイン:**
```html
<p style="margin: 0; color: #2d7a4c; font-weight: bold;">A.</p>
```
→ 緑色で統一

### 2️⃣ 下線色の統一

**❌ 旧デザイン:**
- 赤太字（#e74c3c）
- オレンジ下線（#ff9800）
- マーカー（複数色）

**✅ 新デザイン：**
- 太字のみ（色なし）
- 下線：緑と相性のいい色に統一

**推奨配色:**
- メイン下線：#2d7a4c（濃い緑）
- またはマーク色：#4a90e2（ブルー - 緑と相性良い）

**使用例:**
```html
<!-- パターン1：緑下線 -->
<span style="border-bottom: 3px solid #2d7a4c;">重要な内容</span>

<!-- パターン2：青下線（緑と相性良い） -->
<span style="border-bottom: 3px solid #4a90e2;">ポイント</span>

<!-- パターン3：太字のみ -->
<strong>強調内容</strong>
```

### 3️⃣ 「この記事で分かること」をコンテンツに組み込む

**位置:** 導入段落の直後

**デザイン:**
```html
<!-- 「この記事で分かること」セクション -->
<div style="background-color: #f0f7f0; border-left: 5px solid #2d7a4c; padding: 16px; margin: 20px 0;">
  <p style="font-weight: bold; margin: 0 0 12px 0; color: #2d7a4c;">✓ この記事で分かること</p>
  <ul style="margin: 0; padding-left: 20px;">
    <li>ポイント1</li>
    <li>ポイント2</li>
    <li>ポイント3</li>
    <li>ポイント4</li>
  </ul>
</div>
```

**特徴:**
- ✅ 背景：薄緑（#f0f7f0）
- ✅ 左ボーダー：濃い緑（#2d7a4c）
- ✅ チェックマーク：✓で視覚的強調
- ✅ 見出しテキスト：濃い緑色
- ✅ リストは番号なし（シンプル）
