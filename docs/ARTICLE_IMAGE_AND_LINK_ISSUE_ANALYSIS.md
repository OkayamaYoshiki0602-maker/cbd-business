# 画像・関連記事リンクの問題分析

## 🔍 問題の詳細

### 1. 画像が表示されない

**現状**:
- GitHubのファイル（`wordpress/posts/*.html`）には画像タグが含まれていない
- サイト（https://cbd-no-hito.com/）では画像が表示されている

**原因**:
WordPress REST APIから取得したHTMLには、画像が正しく含まれていない可能性があります。

**確認方法**:
サイトから直接HTMLを取得して、画像タグが存在するか確認します。

---

### 2. 関連記事リンクが消えた

**現状**:
- `cbd-oil-howto.html`には関連記事リンク（`swell-block-postLink`）が存在しない
- バックアップファイルにも存在しない
- しかし、サイトでは関連記事リンクが表示されている

**原因**:
SWELLテーマが自動的に関連記事を生成している可能性があります。

**確認方法**:
サイトから直接HTMLを取得して、関連記事リンクが存在するか確認します。

---

## 📝 対応方法

### 方法1: サイトから直接HTMLを取得（推奨）

WordPress REST APIではなく、サイトから直接HTMLを取得することで、正しい状態を確認できます。

**手順**:
1. サイトから直接HTMLを取得
2. 画像タグと関連記事リンクを確認
3. 必要に応じて、サイトのHTMLをファイルに保存

### 方法2: WordPress管理画面から確認

WordPress管理画面で記事を編集し、画像と関連記事リンクが正しく表示されるか確認します。
