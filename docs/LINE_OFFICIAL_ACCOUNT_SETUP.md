# LINE公式アカウント設定ガイド

## 📋 概要

LINE Messaging API を使用するには、**LINE公式アカウント**が必要です。
公式アカウントを作成することで、Messaging API チャネルが利用可能になります。

---

## 💡 重要なポイント

### ✅ LINE公式アカウントは無料で作成可能

- **無料プラン:** 基本的な機能が無料で使用可能
- **メッセージ送信:** 無料プランでもメッセージ送信が可能（制限あり）
- **個人利用:** 個人でも公式アカウントを作成できます

### 📝 料金について

**無料プラン:**
- メッセージ送信: 無料（月間制限あり）
- 基本的な機能: 無料

**有料プラン:**
- メッセージ送信数が多い場合に必要
- 今回の用途（通知機能）では無料プランで十分

## 🚀 公式アカウント作成後の手順

### Step 1: Messaging API チャネルの確認

1. LINE Developers にログイン
2. プロバイダー「CBD WORLD」を選択
3. 作成した公式アカウントに対応する「Messaging API」チャネルが自動的に作成されているか確認

### Step 2: Messaging API の設定

1. 「Messaging API」タブを開く
2. Channel Access Token を取得
3. `.env`ファイルに設定

---

## ⚠️ 注意事項

### 公式アカウント作成の注意点

- **アカウント名:** 後で変更可能ですが、最初から適切な名前を推奨
- **業種:** 正確に設定してください（後で変更可能）
- **運用目的:** 実際の使用目的に合わせて選択

### 無料プランの制限

- **メッセージ送信:** 無料プランでも可能（月間制限あり）
- **今回の用途:** 通知機能のみなので、無料プランで十分
- **将来的に:** メッセージ送信数が多い場合は有料プランを検討
