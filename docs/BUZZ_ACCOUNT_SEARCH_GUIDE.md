# バズアカウント探しガイド

## 🎯 探すアカウントの条件

- **ジャンル**: ニッチな専門アカウント（健康、美容、投資、技術など）
- **特徴**: アフィリエイトや商品紹介をしている
- **フォロワー数**: 1万人〜10万人
- **分析項目**: ツイート内容、構成、頻度、マネタイズ

---

## 🔍 アカウントの探し方

### 方法1: X (Twitter)の検索機能を使用

1. **キーワード検索**
   - 「アフィリエイト 初心者」
   - 「商品紹介 レビュー」
   - 「健康 サプリ」
   - 「投資 初心者」
   - 「美容 コスメ」

2. **フォロワー数でフィルタ**
   - 検索結果から、フォロワー数が1万〜10万のアカウントを確認
   - プロフィール画面でフォロワー数を確認

3. **関連アカウントから見つける**
   - 見つけたアカウントのフォロー/フォロワーリストを確認
   - 類似アカウントを探す

---

### 方法2: ジャンル別の探し方

#### 健康・サプリ系
- 「CBD サプリ」
- 「健康食品 レビュー」
- 「サプリメント おすすめ」

#### 美容・コスメ系
- 「コスメ レビュー」
- 「美容 おすすめ商品」
- 「スキンケア アフィリエイト」

#### 投資・金融系
- 「投資 初心者」
- 「NISA 解説」
- 「株 おすすめ」

#### 技術・ガジェット系
- 「ガジェット レビュー」
- 「技術 解説」
- 「AI ツール」

---

### 方法3: コミュニティから探す

1. **X (Twitter)のコミュニティ機能**
   - 関連するコミュニティに参加
   - アクティブなメンバーを確認

2. **トレンドから探す**
   - トレンドワードを確認
   - 関連するアカウントを探す

---

## ✅ 選定時のチェック項目

### 1. ツイート内容
- [ ] 専門性が高い内容か
- [ ] アフィリエイトリンクを含むか
- [ ] 商品紹介やレビューがあるか
- [ ] 情報価値が高いか

### 2. ツイート構成
- [ ] 改行が適切に使われているか
- [ ] タイトルや見出しがあるか
- [ ] 読みやすい構成か
- [ ] 絵文字や記号が効果的に使われているか

### 3. ツイート頻度
- [ ] 1日に何回ツイートしているか
- [ ] 定期的にツイートしているか
- [ ] バズっているツイートの頻度はどうか

### 4. マネタイズ方法
- [ ] アフィリエイトリンクの使い方
- [ ] 商品紹介の頻度
- [ ] 広告の使い方
- [ ] 有料コンテンツの宣伝方法

### 5. エンゲージメント
- [ ] いいね数が多いか（フォロワー数の1-5%以上）
- [ ] リツイート数が多いか
- [ ] コメントが多いか
- [ ] 定期的にバズっているか

## 📝 アカウント情報の記録方法

選定したアカウントを以下の形式で記録してください：

```
アカウント名: @username
ジャンル: 健康・サプリ
フォロワー数: 50,000
特徴: CBDサプリのレビューとアフィリエイト
ツイート頻度: 1日3-5回
マネタイズ: アフィリエイトリンク、商品紹介
```

---

## 🔧 分析スクリプトの使用方法

選定したアカウントを `automation/social_media/weekly_buzz_analyzer.py` の `BUZZ_ACCOUNTS` に追加：

```python
BUZZ_ACCOUNTS = [
    'username1',  # 健康・サプリ系
    'username2',  # 美容・コスメ系
    'username3',  # 投資・金融系
    # ... 最大10件
]
```

分析を実行：
```bash
python3 automation/social_media/weekly_buzz_analyzer.py
```

## 💡 参考になるアカウントの特徴

### 良いアカウントの共通点

1. **専門性が高い**
   - 特定のジャンルに特化
   - 詳しい情報を提供

2. **信頼性が高い**
   - 実体験に基づく内容
   - 根拠のある情報

3. **読みやすい**
   - 適切な改行
   - 見出しやリストの活用
   - 絵文字の効果的な使用

4. **マネタイズが自然**
   - アフィリエイトリンクが自然に組み込まれている
   - 商品紹介が押し付けがましくない

5. **エンゲージメントが高い**
   - 定期的にバズる
   - フォロワーとの交流がある

## 📊 分析後の活用方法

分析結果を以下のように活用：

1. **ツイート構成の改善**
   - バズっているツイートの構成を参考
   - 自分のツイートに適用

2. **ツイート頻度の最適化**
   - 効果的な投稿時間を分析
   - 頻度を調整

3. **マネタイズ方法の改善**
   - アフィリエイトリンクの使い方を参考
   - 商品紹介の頻度を調整

4. **コンテンツ戦略の改善**
   - バズっているコンテンツタイプを分析
   - 自分のコンテンツに反映
