# WordPressサイト最適化マスタープラン

## 📋 ゴール

**定量的目標**: 今年中に固定費（約10万円）を回収できるようマネタイズ

**定性的目標**:
- 記事作成がすぐできる
- アフィリエイトリンクをすぐに作成できる
- リンクの差し替えなどもすぐできる
- 画像の管理が容易
- サイトは軽く、画像を軽くしたりできる
- コードは一貫性を持って作られている
- サイト作成の基本を守って導線も綺麗

## 📊 現状分析結果

### コンテンツ統計

- **投稿記事**: 20件
- **固定ページ**: 13件
- **アフィリエイトリンク**: 15件

### 主な改善ポイント

1. **画像のalt属性不足**
   - 多くの記事でalt属性がない画像が存在
   - SEO・アクセシビリティに影響

2. **アフィリエイトリンクの分散**
   - 15件のリンクが複数の記事に分散
   - 一括管理・更新が困難

3. **プラグインの最適化余地**
   - 17個のプラグイン（理想は12-15個）
   - 重複機能のプラグインが存在

## 🎯 実装計画

### Phase 1: 基盤整備（優先度: 高）⭐⭐⭐

#### 1.1 アフィリエイトリンク管理システム

**目的**: リンクの一括管理・更新を容易にする

**実装内容**:
- ✅ アフィリエイトリンク抽出スクリプト（完了）
- ⚠️ スプレッドシートへの出力修正（進行中）
- 📋 スプレッドシートでのリンク管理フォーマット作成
- 📋 WordPress短コードでのリンク埋め込み
- 📋 リンク更新スクリプト（スプレッドシート→WordPress）

**期待効果**:
- リンク作成時間: 5分 → 1分
- リンク更新時間: 10分 → 2分

#### 1.2 画像最適化

**目的**: サイトの軽量化・画像管理の効率化

**現状**:
- Optimoleが有効化されている（自動画像最適化）
- 画像のalt属性が不足している記事が多い

**実装内容**:
- 📋 画像のalt属性追加スクリプト
- 📋 画像ファイル名の命名規則統一
- 📋 画像管理ガイドライン作成

**期待効果**:
- ページ読み込み速度: 改善
- SEOスコア: 向上

#### 1.3 プラグイン最適化

**目的**: サイトパフォーマンス向上

**実装内容**:
- 📋 無効化推奨プラグインの無効化
  - Regenerate Thumbnails（使用後は無効化）
  - WPForms Lite（Contact Form 7を使用中）
  - Perfect Images（Optimoleで代替可能）
  - Blocks Animation（使用頻度が低い場合）

**期待効果**:
- プラグイン数: 17個 → 13個
- ページ読み込み速度: 5-10%改善
