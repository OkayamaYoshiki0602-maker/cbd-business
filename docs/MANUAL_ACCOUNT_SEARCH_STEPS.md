# バズアカウント手動検索手順

## 🎯 検索手順

### Step 1: X (Twitter)にログイン

1. X (Twitter)にログイン
2. 検索バーを使用

---

### Step 2: キーワードで検索

以下のキーワードで検索してください（「ユーザー」タブを選択）：

1. **「アフィリエイト 初心者」**
   - URL: `https://x.com/search?q=アフィリエイト%20初心者&src=typed_query&f=user`

2. **「商品紹介 レビュー」**
   - URL: `https://x.com/search?q=商品紹介%20レビュー&src=typed_query&f=user`

3. **「健康 サプリ」**
   - URL: `https://x.com/search?q=健康%20サプリ&src=typed_query&f=user`

4. **「投資 初心者」**
   - URL: `https://x.com/search?q=投資%20初心者&src=typed_query&f=user`

---

### Step 3: アカウントを選定

検索結果から、以下の条件に合うアカウントを選定：

- ✅ フォロワー数: 1万人〜10万人
- ✅ アフィリエイトリンクを含むツイートがある
- ✅ 商品紹介・レビューをしている
- ✅ 専門性が高い（特定のジャンルに特化）
- ✅ 定期的にツイートしている
- ✅ エンゲージメントが高い（いいね数が多い）

---

### Step 4: アカウント情報を記録

選定したアカウントの情報を記録：

```
アカウント名: @username（@は除く）
ジャンル: 健康・サプリ / 美容・コスメ / 投資・金融 など
フォロワー数: XX,XXX人（検索結果から確認）
特徴: 簡単なメモ
```

---

### Step 5: 検証・分析

#### 方法1: ヘルパースクリプトを使用（推奨）

```bash
# インタラクティブモード（推奨）
python3 automation/scripts/search_accounts_helper.py

# コマンドライン引数で指定
python3 automation/scripts/search_accounts_helper.py username1 username2 username3
```

**インタラクティブモードの流れ:**
1. アカウント名を入力（@は不要）
2. 自動でフォロワー数を検証
3. ジャンルを入力
4. 最大10件まで追加可能
5. 分析を実行するか選択

#### 方法2: 手動で`weekly_buzz_analyzer.py`に追加

```python
# automation/social_media/weekly_buzz_analyzer.py
BUZZ_ACCOUNTS = [
    'username1',  # 健康・サプリ系
    'username2',  # 美容・コスメ系
    'username3',  # 投資・金融系
    # ... 最大10件
]
```

---

### Step 6: 分析を実行

```bash
# 基本分析
python3 automation/social_media/weekly_buzz_analyzer.py

# 高度分析（頻度、マネタイズ、構成を詳細分析）
python3 automation/social_media/weekly_buzz_analyzer.py --advanced
```
