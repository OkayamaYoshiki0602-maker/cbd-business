## 🔍 手動分析のチェック項目

各アカウントについて、以下の項目を確認してください：

### 1. 基本情報
- [ ] フォロワー数（1万〜10万人か）
- [ ] ツイート数
- [ ] ジャンル

### 2. 投稿頻度
- [ ] 1日の平均投稿数
- [ ] 投稿時間帯（効果的な時間）
- [ ] 投稿パターン（毎日/週数回）

### 3. ツイート構成
- [ ] 平均文字数
- [ ] 改行数（段落構成）
- [ ] タイトル記号の使用（【】「」■など）
- [ ] 絵文字の使用率
- [ ] 箇条書きの使用（・→など）

### 4. マネタイズ方法
- [ ] アフィリエイトリンクの使用率
- [ ] 商品紹介の頻度
- [ ] アフィリエイトリンクの配置方法

### 5. ツイート傾向
- [ ] コンテンツタイプ（商品レビュー/情報提供/体験談など）
- [ ] よく使うキーワード
- [ ] バズツイートの特徴

### 6. エンゲージメント
- [ ] 平均いいね数
- [ ] 平均リツイート数
- [ ] エンゲージメント率

## 💡 X APIエラーの解決方法

### 確認事項

1. **X Developer Portalでの設定**
   - App permissions: "Read and write" になっているか
   - User authentication settings: 適切に設定されているか

2. **API認証情報**
   - Access Token と Access Token Secret を再生成
   - `.env`ファイルに正しく設定されているか

3. **APIプラン**
   - 無料プランで他のユーザーのツイートを取得できるか確認
   - 必要に応じてプランをアップグレード
