# ツイート生成改善サマリー

## ✅ 改善内容

### 1. 記事ツイートの改善

**問題点**:
- タイトルのみで、記事の内容が分からない
- 引き付けがない

**改善内容**:
- 記事の要約を必ず含める
- 要約は見やすい形で、記事の要点を簡潔に伝える
- 読者が「この記事を読みたい」と思うような魅力的な要約
- 具体的な情報や数字を含める
- 記事の価値を明確に伝える

**ツイート構成**:
```
【タイトル】

記事の要約（2-3文程度、魅力的に）

[URL]
```

---

### 2. ニュースツイートの改善

**問題点**:
- 英語のタイトルがそのまま表示される
- 記事の要約がない
- 引き付けがない

**改善内容**:
- **タイトルを自然な日本語に翻訳**
- **ニュースの要約を必ず含める**
- **要約に以下の要素を含める:**
  - 日本と海外のギャップ（海外の成熟度と日本の現状のギャップ）
  - 常識が変わるような意外性
  - 実は〇〇の効果があったような研究や論文
  - 定量的なCBDの体への効果
  - 政治（法律の改正など）への影響
  - 経済への影響
- エビデンスとなる数字やデータを明確に記載
- 結局何が言いたいかを明確にする

**ツイート構成**:
```
【タイトル（日本語）】

ニュースの要約（2-4文程度、上記の要素を含めて魅力的に）

[URL]
```

## 📝 改善されたプロンプトの要点

### 記事ツイート（実体験×データ分析型、ライフスタイル型）

- 記事の要約を必ず含める
- 要約は見やすい形で、記事の要点を簡潔に伝える
- 読者が「この記事を読みたい」と思うような魅力的な要約
- 具体的な情報や数字を含める

### ニュースツイート（その他型、データ分析型）

- タイトルは英語の場合、自然な日本語に翻訳
- ニュースの要約を必ず含める
- 要約に以下の要素を含める:
  - 日本と海外のギャップ
  - 意外性
  - 研究・論文
  - 定量的な効果
  - 政治・経済への影響
- エビデンスとなる数字やデータを明確に記載
- 結局何が言いたいかを明確にする
