## 🚀 ワークフロー

### 完全な記事生成フロー

```
【Step 1】Google Sheets で記事テーマを入力
┌─────────────────────────────────────────┐
│ A列: タイムスタンプ（自動）              │
│ B列: ステータス「新規」                  │
│ C列: 記事タイトル                       │
│ D列: 記事の分類（15個のテンプレート）   │
│ E列: ターゲット                         │
│ F列: タグ                               │
│ G～I列: 自動入力箇所                     │
└─────────────────────────────────────────┘

【Step 2】タイトル候補を生成
$ python3 generate_title_options.py --generate

結果:
├─ K～O列: 5つのタイトル候補が自動入力
├─ ステータス: 「タイトル選択待ち」に変更
└─ ユーザーに通知

【Step 3】ユーザーがタイトルを選択
Google Sheets で:
├─ K～O列のタイトル候補を確認
├─ P列に最適なタイトルをコピー&ペースト
└─ B列を「生成待ち」に変更

【Step 4】記事を自動生成
$ python3 article_generator_html_v2.py

処理内容:
├─ P列から選択されたタイトルを優先読み込み
├─ 15個のテンプレートから最適なものを選択
├─ メタデータを自動生成
├─ SEOスラッグを自動生成
├─ HTML 記事を生成（2,500-3,500文字）
├─ 関連記事を自動推奨（3件）
└─ WordPress に下書き投稿

【Step 5】WordPress で確認・編集・公開
└─ デフォルトステータス: 下書き（編集可能）

【結果】
✅ ユーザーが選択したタイトル
✅ 適切なテンプレートで生成
✅ メタデータ完備
✅ SEO最適化済み
✅ 関連記事リンク付き
✅ 高品質な記事
```
