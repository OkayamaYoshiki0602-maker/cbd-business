# CBD記事テンプレート案（5種類）

## 概要
記事の特性ごとに最適化されたHTMLテンプレートを使用し、高精度で多様な記事を生成。

---

## テンプレート1: 商品レビュー型（Naturecan, roun など）

### 特徴
- **目的**: ブランドの信頼性・品質・理由を徹底解説
- **構成**: ブランド背景 → 品質管理 → 商品紹介 → 推奨者像
- **対象**: 高級・信頼性重視の商品
- **記事例**: 1085（初心者ガイド）、1097（Naturecan）

### 構成要素
1. **導入** (共感→問題→結論)
   - 「失敗したくない」「1ミリの不安も残したくない」→「結論は○○一択」
   
2. **この記事で分かること** (3-4項目)
   
3. **ブランド背景・信念** (H3)
   - 創設者の背景、ブランドの哲学
   - 画像（ブランドロゴやイメージ）
   
4. **品質管理・強み** (H3)
   - 検査プロセス、テスト段階
   - テーブル：複数行の詳細情報
   
5. **おすすめ商品** (H3)
   - H4で各商品を紹介（🥇🥈など使用）
   - 商品テーブル（項目：内容）
   - メリット/デメリット/おすすめ対象
   - アフィリエイトボタン
   
6. **まとめ** (H3)
   - チェックリスト形式で推奨者像
   - 最後に「安全をお金で買う」的なメッセージ
   
7. **参考文献** (H3)

### 文字数目安
- **全体**: 3,000-3,500文字
- **導入**: 150-200文字
- **セクション別**: 250-400文字

### 適用記事
- 高級ブランド商品紹介
- ブランド比較
- 信頼性重視のユーザー向け

---

## テンプレート2: 初心者ガイド型（選び方・使い方）

### 特徴
- **目的**: CBDを初めて使う人に必要な全知識を提供
- **構成**: 基本知識 → 選び方 → 使い方 → FAQ
- **対象**: CBD初心者全般
- **記事例**: 「CBD初心者ガイド：選び方から使い方まで」

### 構成要素
1. **導入** (2-3段落)
   - 疑問形：「CBDって何？」「どれを選べばいいの？」
   
2. **この記事で分かること** (4-5項目)
   
3. **基本知識** (H3)
   - 定義、THCとの違い
   - メリット/デメリットボックス
   
4. **商品ベスト3** (H3)
   - H4で各商品（🥇🥈🥉使用）
   - 各商品：テーブル + メリット + おすすめ対象 + ボタン
   
5. **選び方3つのポイント** (H3)
   - H4で各ポイント（「摂取方法」「濃度」「価格」など）
   - テーブルで比較情報
   
6. **使い方ステップガイド** (H3)
   - H4で各ステップ（Step 1, 2, 3...）
   - チェックリスト形式

7. **よくある質問（FAQ）** (H3)
   - swell-block-faqクラス使用
   - 5-8個のQ&A
   
8. **まとめ** (H3)
   - チェックリスト：3つの大事なポイント
   
9. **参考文献** (H3)

### 文字数目安
- **全体**: 3,500-4,500文字（詳細ガイド）
- **各セクション**: 200-400文字

### 適用記事
- 初心者向けガイド
- 「選び方」記事
- 「使い方」記事
- 比較・ランキング記事

## テンプレート3: 悩み解決型（睡眠、ストレス、集中など）

### 特徴
- **目的**: 特定の悩みを解決する手段としてのCBDを提案
- **構成**: 悩みの背景 → 原因・メカニズム → CBDの役割 → 実践方法
- **対象**: 特定の課題を抱えるユーザー
- **記事例**: 「CBD睡眠効果ガイド」「ストレス緩和」

### 構成要素
1. **導入** (2-3段落)
   - 悩みへの共感：「眠れない」「ストレスが…」
   - 問題の深刻性を提示
   - CBDが解決策になることを示唆
   
2. **この記事で分かること** (3-4項目)
   
3. **悩みの背景・原因** (H3)
   - メカニズム説明（例：自律神経のバランス）
   - 統計データ、研究情報
   
4. **CBDのメカニズム** (H3)
   - ECS（エンドカンナビノイドシステム）説明
   - 括弧で専門用語を解説
   - 図表あるとベター
   
5. **CBDの実証と研究** (H3)
   - 研究論文、臨床結果
   - テーブル：複数の研究結果比較
   
6. **おすすめCBD製品** (H3)
   - H4で2-3個の製品紹介
   - 悩み別に「このタイプがおすすめ」と説明
   
7. **実践方法・ステップ** (H3)
   - 摂取量、摂取タイミング、期間
   - チェックリスト：効果測定方法
   
8. **注意点・副作用** (H3)
   - 医学的注意事項（薬機法遵守）
   - 「医師に相談してください」も忘れずに
   
9. **まとめ** (H3)
   - 悩み解決のための3ステップ
   
10. **参考文献** (H3)

### 文字数目安
- **全体**: 3,000-3,800文字
- **各セクション**: 200-350文字

### 適用記事
- 睡眠記事
- ストレス緩和記事
- 集中力UP記事
- リラックス記事

## テンプレート4: 経済・ビジネス型（市場トレンド、法規制）

### 特徴
- **目的**: CBD産業・市場・法規制に関する情報提供
- **構成**: 現状 → トレンド → 経済影響 → 投資視点
- **対象**: CBD業界に興味のあるユーザー、投資家
- **記事例**: 「2026年CBD市場トレンド」「大麻法制化」

### 構成要素
1. **導入** (2-3段落)
   - 市場規模、成長率、背景
   - 日本での状況を強調
   
2. **この記事で分かること** (3-4項目)
   
3. **市場規模・成長率** (H3)
   - グラフデータ（テーブルで数字）
   - 世界 vs 日本での比較
   
4. **トレンド分析** (H3)
   - 複数のトレンド（各H4）
   - テーブル：トレンド項目 × 説明
   
5. **法規制・政策** (H3)
   - 日本の現状
   - 海外との比較
   - 今後の見通し
   
6. **ビジネスチャンス** (H3)
   - 企業動向
   - 参入企業、スタートアップ
   
7. **消費者への影響** (H3)
   - 価格、選択肢、品質への影響
   - メリット/デメリット
   
8. **まとめ** (H3)
   - 今後の予測
   - ユーザーにとっての意味
   
9. **参考文献** (H3)
   - 公式統計、業界レポート、政府資料

### 文字数目安
- **全体**: 2,500-3,500文字
- **各セクション**: 150-300文字

### 適用記事
- 市場分析記事
- 法規制記事
- トレンド予測
- ビジネス記事

## テンプレート5: 製品比較型（複数商品の比較表）

### 特徴
- **目的**: 複数の同カテゴリー商品を客観的に比較
- **構成**: 比較基準 → 詳細比較 → メリット・デメリット → 選定ガイド
- **対象**: 複数商品を迷っているユーザー
- **記事例**: 「CBDオイル5製品比較」「グミ製品ランキング」

### 構成要素
1. **導入** (2-3段落)
   - 選択の難しさを認識
   - 「この記事で比較します」と提示
   
2. **この記事で分かること** (3-4項目)
   
3. **比較基準の説明** (H3)
   - 何を比較するのか（価格、濃度、成分など）
   - なぜそれが大事か
   
4. **一覧比較テーブル** (H3)
   - 大きなテーブル：商品 × 項目
   - 複数行で全製品の情報を網羅
   
5. **個別製品詳細** (H3)
   - H4で各製品（🥇🥈など）
   - 製品テーブル + メリット + デメリット + ボタン
   
6. **カテゴリー別おすすめ** (H3)
   - 「初心者に最適」「コスパ重視」など
   - チェックリスト：対象者別おすすめ
   
7. **選定ガイド** (H3)
   - どの製品をいつ選ぶか
   - フローチャート的な説明
   
8. **まとめ** (H3)
   - 最終的なおすすめ方針
   - 「複数試す価値」も説明
   
9. **参考文献** (H3)

### 文字数目安
- **全体**: 3,500-4,500文字（比較が多いため）
- **各詳細**: 250-350文字

### 適用記事
- 商品比較記事
- ランキング記事
- カテゴリー別ガイド
- 「どれを選ぶ」記事

## メリット

1. **記事品質の統一**: テンプレートに従うことで常に高品質を維持
2. **記事タイプの多様性**: 5種類で様々なニーズに対応
3. **自動化の加速**: テンプレート切り替えで週10-20記事の生成も可能
4. **ユーザー満足度UP**: 異なる視点の記事が増える
5. **SEO最適化**: 各テンプレートで異なるキーワード + 内部リンク機会増加

---

## 注意点

1. **医療効果の表現**: 全テンプレートで「効果の個人差」を必ず記載
2. **アフィリエイト表示**: 記事冒頭に免責事項を記載
3. **情報の信頼性**: テンプレート3・4は特に「公式情報」「研究結果」を優先
4. **文字数管理**: 各テンプレートの目安文字数を厳守
