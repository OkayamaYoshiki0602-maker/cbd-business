## 🔍 無料アカウントは複数使えるのか？

### 答え: **複数のアプリ/アカウントで使用可能**

**理由:**
1. **無料プランは開発者アカウント単位**
   - 1つの開発者アカウントで複数のアプリを作成可能
   - 各アプリごとに投稿制限がある可能性

2. **別アカウントでも使用可能**
   - 既存のアカウント（CBD関連）で無料プランを使用中
   - 新しいアカウント（@otonasan_review）でも別のアプリとして作成可能
   - **ただし、投稿制限は開発者アカウント全体で共有される可能性がある**

### 確認方法

1. **X Developer Console → 「使用状況」タブを確認**
   - 現在の使用状況を確認
   - 投稿制限が「開発者アカウント単位」か「アプリ単位」かを確認

2. **「Projects & Apps」タブを確認**
   - 複数のアプリを作成できるか確認
   - 各アプリの投稿制限を確認

---

## ⚠️ 注意事項

### 1. 投稿制限の共有

**可能性A: 開発者アカウント単位で制限が共有される**
- 既存アカウント + 新規アカウント = 合計100投稿/月
- この場合、両方のアカウントで投稿すると制限に達しやすい

**可能性B: アプリ単位で制限がある**
- 各アプリごとに100投稿/月
- この場合、それぞれ独立して投稿可能

### 2. 制限の確認方法

**X Developer Consoleで確認:**
1. 「Dashboard」タブ → 各プロジェクト/アプリの投稿制限を確認
2. 「使用状況」タブ → 全体の使用状況を確認
3. 「Projects & Apps」タブ → 各アプリの制限を確認
