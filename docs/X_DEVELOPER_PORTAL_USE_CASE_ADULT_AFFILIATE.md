## Xのデータと API の使用例説明（日本語）

### 1. プロジェクト概要

**プロジェクト名:** アダルトコンテンツ情報提供・アフィリエイト自動投稿システム

**目的:** アダルト動画コンテンツに関する新作情報を、X（旧Twitter）で適切に発信し、ユーザーに最新情報を提供するための自動化システムを構築します。DMMアフィリエイトプログラムを通じて、合法的なアフィリエイト活動を行います。

**重要な注意事項:**
- 18歳未満の方の閲覧を防止するため、プロフィールに明確なR18表記を掲載
- Xの利用規約・開発者規約を遵守
- 過度に露骨な画像は使用せず、公式サンプル画像のみを使用
- アフィリエイトリンクを含むことを明記

---

### 2. 具体的な使用例

#### 2.1 新作アダルト動画情報の自動投稿

**目的:**
DMMアフィリエイトAPIから取得した新作アダルト動画の情報を、Xで自動的に共有し、ユーザーに最新情報を提供します。

**実装方法:**
1. **商品情報の取得**
   - DMMアフィリエイトAPIを使用して、新作アダルト動画の商品情報を取得
   - 取得する情報: タイトル、女優名、発売日、公式サンプル画像URL、アフィリエイトリンク

2. **ツイート文案の生成**
   - 取得した商品情報から、280文字以内のツイート文案を自動生成
   - テンプレートを使用して、統一感のある投稿を作成
   - 必ずR18表記（🔞）とハッシュタグ（#AV #エロ動画 #R18）を含める

3. **画像の処理**
   - 公式サンプル画像をダウンロード
   - 画像サイズをTwitter推奨サイズに調整（必要に応じて）
   - 過度に露骨な画像は使用せず、公式サンプル画像のみを使用

4. **投稿の実行**
   - 画像をTwitter APIにアップロード
   - ツイート文案と画像を組み合わせて投稿
   - 投稿前に内容を確認する仕組みを実装（オプション）

**使用するAPI:**
- `POST /2/tweets` - ツイート投稿
- `POST /1.1/media/upload` - 画像アップロード（公式サンプル画像）

**投稿頻度:**
- 平日: 1日2回程度（13:00、20:00など）
- 休日: 1日3回程度（11:00、15:00、21:00など）
- スパム行為は行わない
- レート制限を遵守

**コンテンツ:**
- 作品タイトル
- 女優名
- 発売日
- 公式サンプル画像（1枚）
- アフィリエイトリンク（DMMアフィリエイトプログラム経由）
- ハッシュタグ（#AV #エロ動画 #R18）
- R18表記（🔞）

---

#### 2.2 人気作品・おすすめ作品の紹介

**目的:**
ユーザーが興味を持ちそうな人気作品やおすすめ作品を、定期的に紹介します。

**実装方法:**
- DMMアフィリエイトAPIから、人気作品や特定ジャンルの作品を検索
- 作品情報を取得し、紹介文を含むツイートを生成
- 公式サンプル画像と共に投稿

**投稿頻度:**
- 週2-3回程度
- 過度な投稿は避ける

---

### 3. データの使用方法

**取得するデータ:**
- ツイート投稿結果（投稿ID、URLなど）
- ツイートのエンゲージメント（いいね数、リツイート数など） - 将来的に分析用

**データの用途:**
- 投稿履歴の記録（重複投稿の防止）
- エンゲージメント分析（投稿タイミングの最適化）
- アフィリエイトリンクのクリック率分析

**データの保存・共有:**
- 投稿履歴はローカル環境のみに保存
- 外部にデータを再販売することは一切ありません
- ユーザーのプライバシーを尊重し、個人情報の収集は行いません
- X APIから取得したデータを再販売することはありません

---

### 4. コンプライアンスと規約遵守

**遵守する事項:**
- ✅ Xの利用規約・開発者規約を遵守
- ✅ スパム行為は行わない（過度な投稿を避ける）
- ✅ 自動リプライや自動メンションは行わない
- ✅ 18歳未満の方の閲覧を防止するため、プロフィールに明確なR18表記を掲載
- ✅ 過度に露骨な画像は使用せず、公式サンプル画像のみを使用
- ✅ アフィリエイトリンクを含むことを明記
- ✅ レート制限を遵守
- ✅ ユーザーのプライバシーを尊重

**コンテンツポリシー:**
- 公式サンプル画像のみを使用（無断転載は行わない）
- モザイク処理済みの画像のみを使用
- 過度に露骨な表現は避ける
- 誤解を招く表現は使用しない

**自動化の範囲:**
- 商品情報の取得、ツイート文案生成、画像処理、投稿予約までは自動化
- 投稿内容は定期的に確認し、不適切な内容は投稿しない
- 必要に応じて、投稿前に人間が確認する仕組みを実装

### 5. 商業利用について

**商業目的:**
- DMMアフィリエイトプログラムを通じた合法的なアフィリエイト活動
- アフィリエイトリンクからの手数料収入

**収益モデル:**
- DMMアフィリエイトプログラム経由のアフィリエイトリンクから手数料を得る
- X APIを使用した直接的な収益化は行わない
- Xのデータを再販売することは一切ありません

**アフィリエイトリンクの扱い:**
- DMMアフィリエイトプログラムの規約を遵守
- アフィリエイトリンクを含むことをプロフィールに明記
- 誤解を招く表現は使用しない

### 6. 技術的な実装

**開発環境:**
- Python 3.9+
- tweepy ライブラリ（X API v2）
- dmm-search3 ライブラリ（DMMアフィリエイトAPI）
- Pillow（画像処理）

**運用方法:**
- ローカル環境またはプライベートサーバーで実行
- GitHubでコード管理
- 定期的なメンテナンスと更新

**セキュリティ:**
- API認証情報は環境変数で管理（.envファイル）
- 認証情報はGitにコミットしない
- 定期的なセキュリティチェック
