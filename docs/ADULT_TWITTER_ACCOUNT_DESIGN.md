# 18禁アフィリエイトTwitterアカウント設計

## 📊 どちらが簡単か？

### ビデオアカウントの方が簡単（推奨）✅

**理由**:
1. **コンテンツ作成が簡単**
   - サムネイル画像 + 短い紹介文で十分
   - 漫画のように複数のコマを選ぶ必要がない
   - 感想・紹介文が短くて済む

2. **アフィリエイト単価が高い**
   - ビデオは商品単価が高い（1,000円〜5,000円）
   - 報酬率も高い（30-50%）
   - 少ない成約数でも収益化しやすい

3. **検索需要が安定**
   - 女優名、メーカー名、ジャンルなど検索パターンが明確
   - 新作情報の需要が高い

4. **コンテンツ管理が簡単**
   - 作品情報（タイトル、女優、発売日）が明確
   - レビュー要素を追加しやすい

---

### 漫画アカウントの難しさ

1. **画像選定が難しい**
   - どのコマを選ぶか判断が必要
   - 複数のコマを組み合わせる必要がある可能性

2. **コンテンツ作成に時間がかかる**
   - ストーリー性を考慮する必要がある
   - ネタバレにならないように注意が必要

3. **単価が低め**
   - 同人誌は500円〜1,500円程度
   - 報酬率は20-40%

---

## 🎯 ビデオアカウント設計

### アカウント名（ユーザー名）

**候補1（推奨）**: `@AV_review_○○` （○○は数字や記号）
- 例: `@AV_review_info`
- シンプルで検索性が高い

**候補2**: `@動画情報○○`
- 例: `@動画情報_18`
- 日本語で分かりやすい

**候補3**: `@新作AV情報○○`
- 例: `@新作AV情報_2025`
- 明確に新作情報を発信することを示す

**推奨**: `@AV_review_info` または `@AV_info_18`
- 短くて覚えやすい
- 検索性が高い
- 18禁であることが分かる

---

### 表示名（プロフィール名）

**候補1（推奨）**: 
```
新作AV情報🔞 | 18歳未満閲覧禁止
```

**候補2**: 
```
AV新作レビュー🔞 R18
```

**候補3**: 
```
動画情報発信🔞 | 新作・人気作品
```

**推奨**: `新作AV情報🔞 | 18歳未満閲覧禁止`
- 明確にR18表記
- 18歳未満閲覧禁止を明記（コンプライアンス）
- 新作情報を発信することを明示

---

### プロフィール説明（自己紹介）

**推奨テンプレート**:

```
🔞 18歳未満の方の閲覧・フォローはご遠慮ください

📹 新作AV情報を発信
💬 人気作品・おすすめ作品を紹介
🎯 最新情報をいち早くお届け

#AV #エロ動画 #R18

※アフィリエイトリンクを含みます
```

**要点**:
- ✅ 最初に「18歳未満閲覧禁止」を明記
- ✅ 何を発信するかを明確に
- ✅ アフィリエイトリンクを含むことを明記（透明性）
- ✅ ハッシュタグで検索性向上

---

### プロフィール画像

- 控えめなアイコンを使用
- 過度に露骨な画像は避ける
- Twitter規約に違反しない範囲で

**例**:
- シンプルなロゴやアイコン
- 「R18」や「AV情報」などのテキスト画像
- モザイク処理済みの控えめな画像

---

### ピン留めツイート

初期のピン留めツイート例:

```
🔞 18歳未満の方の閲覧・フォローはご遠慮ください

このアカウントでは新作AV情報を発信しています。
アフィリエイトリンクを含む場合があります。

フォローありがとうございます🙏
```

## ⚠️ コンプライアンス確認

### DMMウェブサービス利用規約の重要ポイント

1. **第8条（2）**: DMMへのリンクを設置しなければならない
   - ✅ アフィリエイトリンクはDMMへのリンクなのでOK

2. **第9条（1）-3**: 不潔またはグロテスクなコンテンツ等一般人が不快感を覚える内容
   - ⚠️ 過度に露骨な画像は避ける
   - ✅ サムネイル画像程度であれば問題なし

3. **第11条**: クレジット表示
   - ✅ プロフィールに「アフィリエイトリンクを含みます」と明記（実質的なクレジット表示）

4. **第9条（1）-7**: 閲覧者に誤解を与えるおそれのある内容
   - ✅ プロフィールでアフィリエイトであることを明記
   - ✅ 商品情報を正確に表示

## 📋 DMMアフィリエイトサイト登録時の入力内容

### サイト追加申請フォーム

**サイト名**: 
```
AV新作情報アカウント
```
または
```
新作AV情報発信
```

**サイトURL**: 
```
https://twitter.com/[アカウント名]
```
例: `https://twitter.com/AV_review_info`

**注意**: 
- サイトURLには必ずユーザー名を含む完全なURLを入力
- 例: `https://twitter.com/○○○○` の形式
