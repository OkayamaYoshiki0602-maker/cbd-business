## 🚀 システム全体像

### 記事生成フロー（全自動）

```
【毎朝8:00】
┌─────────────────────────────────────┐
│ シート2（記事テーマ）から入力を読込 │
├─────────────────────────────────────┤
│ - ステータス: 新規
│ - 記事テーマ: ユーザー指定
│ - 記事の分類: 10種類から選択
│ - 使用アフィリエイト: prod_001等
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│ Gemini APIで記事生成                │
├─────────────────────────────────────┤
│ - テンプレート自動選択
│ - HTML直接出力（高品質）
│ - 各テンプレートの完全カスタマイズ
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│ WordPressに下書き投稿              │
├─────────────────────────────────────┤
│ - 記事ID取得
│ - メタデータ設定
│ - アフィリエイトリンク挿入
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│ スプレッドシート自動反映            │
├─────────────────────────────────────┤
│ - シート1（記事一覧）: 記事情報追記
│ - シート3（アフィリエイト管理）:
│   K列に記事ID追記
│   L列に記事タイトル追記
│   M列（使用記事数）を再計算
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│ LINE通知でプレビュー送信            │
├─────────────────────────────────────┤
│ - 記事プレビュー
│ - 確認・編集のお願い
└─────────────────────────────────────┘
           ↓
     【ユーザー確認・編集】
     （WordPress管理画面）
           ↓
    スプレッドシート（シート2）で
    「承認済み」に変更
           ↓
【13時、18時、21時】
┌─────────────────────────────────────┐
│ 自動投稿タイミング                  │
├─────────────────────────────────────┤
│ - 承認済み記事を検出
│ - WordPressに公開投稿
│ - SNS連携（X自動投稿）
│ - ステータスを「投稿済み」に更新
└─────────────────────────────────────┘
```

## 📊 アフィリエイト管理の革新

### 従来（商品ごと管理）
```
記事A → Naturecan
記事B → Naturecan（重複管理）
記事C → roun
```
❌ 商品情報が分散  
❌ 成績追跡が困難  
❌ 重複入力が必要

### 新システム（シート3 で一元管理）
```
【シート3: アフィリエイト管理】
prod_001: Naturecan
  └─ 使用記事: 記事A, 記事B, 記事D
  └─ 使用記事数: 3
  └─ CV数: 5
  └─ 売上: ¥50,000

prod_002: roun
  └─ 使用記事: 記事C
  └─ 使用記事数: 1
  └─ CV数: 2
  └─ 売上: ¥10,000
```

✅ 商品ごとに一元管理  
✅ 複数記事で再利用可能  
✅ 成績が可視化  
✅ 最適な商品選択が可能

```

   → CVR: 3-5% ← 最高CVR期待

### 月間売上シミュレーション

```
【保守的推定】
- 平日: 毎日1記事投稿 × 22日 = 22記事

- 月計: 30記事

各記事の平均CVR: 2.5%
各CVの平均成約金: ¥1,000

月間CV: 30記事 × 30訪問/日 × 2.5% = 約23個

月間CV: 50 × 30 × 3% = 約45個
月間売上: 45 × ¥1,500 = ¥67,500

【実現目標】
固定費: 約¥11,000/月（Cursor ¥10,000 + サーバー ¥1,000）
